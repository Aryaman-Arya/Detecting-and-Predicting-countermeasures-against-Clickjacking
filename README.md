# Detecting and Predicting countermeasures against Clickjacking

The Clickjacking attack involves creating a malicious web page to deceive the user into making unwanted clicks that benefit the attacker. It spreads malware, steals passwords and cookies, transmits spam, and deletes personal emails, among other things. Clickjacking attacks occur when many programmes or websites share a graphical display: A principal can trick the user into interacting with user interface elements owned by another principal, initiating actions that the user did not expect.
